Skill Class

Kegiatan belajar bersama dengan sistem sharing oleh HMTI yang membahas tentang keahlian tingkat dasar hingga mahir terkait Teknologi Informasi. Meningkatkan pengetahuan dan keterampilan anggota HMTI dalam profesionalitas bidang Teknologi Informasi.

HMTI Fun

Tidak selalu di depan komputer, kadang kita butuh bersenang-senang. Karena itu HMTI menyediakan HMTI Fun sebagai sarana untuk melakukan kegiatan di luar IT seperti Futsal dan E-Sport.

March Event

HMTI mengadakan kegiatan tahunan bertajuk 'March Event' yang di dalamnya diadakan beberapa mata lomba terkait IT dengan tujuan meningkatkan kompetisi dan daya saing para pelajar dan mahasiswa demi peningkatan kualitas Sumber Daya Manusia.

IT Expo

Expo sebagai ajang untuk memamerkan inovasi oleh para mahasiswa Teknik Informatika Universitas Khairun. Sebagai sarana sosialisasi produk-produk IT sekaligus tolak ukur kualitas Sumber Daya Manusia HMTI.
